The Application

· 2020-06-15 · # 原创 # 沉渊梦舟录

“你想去北方吗？”
顾忮心拖过大敞着身子的志愿填报参考资料，目光顺着一列“北”字扫下去，在几个被铅笔圈住的名字上稍作停留。而林木秀瘫在床上，用手臂遮住眼睛，懒懒地答了一句看你意思。
顾忮心拿起铅笔，皱着眉头，来回点了点两个名字。笔尖最后定定地压在了上面的那一个上。他转过头来，林木秀还躺在那里，不过已经翻了个半个身子，正侧向他，两只眼睛探寻着他脸上的答案。
有了。顾忮心把笔一抛而下，起身走到床沿，抬起膝盖拱了拱床上的人。林木秀会意地向后挪了挪身子，让出一个人的位置来。顾忮心欣然躺下，两个人面对着面，一时无言。
直到顾忮心凑上前来，用自己的嘴唇轻轻碰了碰林木秀的。
“S大的中文就挺好的。去年省排三百多就可以上，即使是实验班也在两百开外，以我们的成绩应该没问题。”顾忮心垂着眼睛，一只手漫不经心地玩着林木秀半垮的腰带。
果不其然，下一刻他的手就被捉住了。
“但实验班在我们省只招两个人，”林木秀和他五指交扣，大拇指轻轻摩挲着他的。“这几年投档线的排名一直在上升，只要有一个人比我们考得高，我们就会分开。”
“哪怕这样，后面也还有分流呢。我们即使分开，也就一年到顶。”
“但我们住的寝室会不一样，上的课也是，”林木秀闭上眼睛。“我不是很想错过。”
即使是你生命中的区区一年，我也不想错过。顾忮心的喉咙发干，这让他下意识地咽了口口水。“那你的意思是？”
林木秀睁开眼睛。
“我只是把我的感受说给你听，没有别的意思。”他微微一笑。“非要说的话，我希望你还是照着自己的想法去做。如果顾虑我的感受来做决定，最后结果不尽如人意，倒还不如在先前就选一条让自己不会后悔的道路。”
“后不后悔可不是你和我能说了算的。”顾忮心也笑了。“但还是谢谢你，这话挺让人开心的。”
